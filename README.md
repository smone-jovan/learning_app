# Learning App

Flutter gamified learning app with Firebase backend.

## Features
- User authentication
- Quizzes and courses
- Gamification (points, coins, levels)
- Leaderboard
- Achievements
- Real-time updates

## Tech Stack
- Flutter
- Firebase (Auth, Firestore, Storage)
- GetX (State Management)

## Setup
1. Clone repo
2. Run `flutter pub get`
3. Add your `google-services.json` (Android) and `GoogleService-Info.plist` (iOS)
4. Run `flutter run`
